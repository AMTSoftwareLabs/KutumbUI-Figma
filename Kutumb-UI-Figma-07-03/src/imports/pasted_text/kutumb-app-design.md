Project Name: Kutumb (Comprehensive Multi-functional Family Management App)
Design System/Style: Material Design 3 (M3), edge-to-edge layout, playful yet organized, generous padding, rounded corners, uses dynamic theming (Seed Color: #4CAF50 or similar vibrant green/earthy tones). Include a Floating Action Button (FAB) on primary screens.
Global Elements (Present on almost all screens):
Top App Bar: Shows the Screen Title (e.g., "Kutumb", "Vyaya"), a user avatar or profile icon on the right, and occasionally a "Loyalty Score" badge (a small leaf/star icon with a number like "850 pts").
Bottom Navigation Bar (Scrollable or standard layout): 8 distinct tabs with labels and icons:
Home (Kutumb) - House/Home icon
Rules/Tasks (Niyama) - Checkbox/Check icon
Chat (Samvaad) - Chat bubble/Email icon
Expenses (Vyaya) - Shopping Cart icon
Loans (Rina) - Money/Bank icon
Lists (Soochi) - Bullet list icon
Memories (Smriti) - Photo Album/Image icon
Profile (Parichay) - User/Account icon
1. Home Screen (Kutumb - Dashboard)
Hero Section: A warm greeting ("Namaste, [Name]!") and a summary subtitle ("Here is your daily household overview").
Quick Stat Cards (Horizontal scroll or Grid):
Nirantarta (Streak): Fire icon, "12d", subtitle "Nirantar karam" (Active streak).
Niyam (Active Rules): Check icon, "4", subtitle "Sakriya niyam".
Vyaya (Expenses): Cart icon, "₹14,500", subtitle "30 din mein kharch" (Past 30 days).
Team/Household Score Card: A large gamified progress or gauge chart showing the "Household Sync Score" (e.g., 92/100).
Daily AI Tip: A styled highlight card with a lightbulb icon showing a dynamically generated household tip.
Recent Activity Feed: A vertical list showing recent events (e.g., "Ravi marked groceries as done", "Priya added ₹500 for milk").
2. Rules & Tasks Screen (Niyama)
Top Sub-Navigation (Tab Row): 4 tabs -> Niyam (Rules), Karya (Tasks), Puraskar (Rewards), Dand (Punishments).
Tab 1: Niyam (Rules): List of active household rules (e.g., "No phones at dinner"). Each card has a rule title, consequence tag, and an "Apply Penalty" button.
Tab 2: Karya (Tasks): Daily/Weekly chores checklist. Checkboxes next to items (e.g., "Take out trash", "Do dishes").
Tab 3: Puraskar (Rewards): List of unlockable rewards (redeemable with points). Shown as gift cards with point costs (e.g., "Movie Night - 500 pts"). Redeem button.
Tab 4: Dand (Punishments): List of configured penalties.
Primary Action: A prominent Floating Action Button (FAB) with a '+' icon to add a new Rule, Task, Reward, or Punishment based on the active tab.
3. Chat Screen (Samvaad)
Chat Interface: Standard conversational UI.
Message Bubbles: Right-aligned (user's messages) in the primary color, Left-aligned (partner/family) in surface-variant color. Timestamps inside bubbles.
Input Box (Footer): Text field reading "Type a message...", an attachment icon (paperclip/image), and a send button (paper airplane icon).
4. Expenses Screen (Vyaya)
Top Section (Overview):
Two large hero metrics: Total Expenses ("₹24,500") and Total Shared Income ("₹80,000").
Settle Up / Split view: A horizontal comparison bar showing "User A Paid: ₹15,000" vs "User B Paid: ₹9,500" with a balancing arrow in the middle.
Budget Progress Bar: "Monthly Spending Limit" text, a progress bar filling up, and text "₹24k / ₹30k". Changes to amber/red if nearing the limit.
Breakdown Section: Category list (Groceries, Rent, Utilities) with mini progress bars showing category limits.
Transaction List: Recent expenses with date, category icon, user who paid, and amount.
Primary Action (FAB): '+' icon to log a new expense or income.
Add Expense Modal (Popup mockup): Form containing: Amount (Numpad input), dropdown for Category, Date picker, Note text field, toggle for "Expense" vs "Income", and a "Save" button.
5. Loans & Debts Screen (Rina)
Top Sub-Navigation: 2 tags -> Drishya (Overview), Sakriya Rina (Active Loans).
Overview Tab (Drishya):
Total Active Debt ("Kul Sakriya Rina") showing the large remaining principal.
Estimated EMI ("Anumanit EMI") showing monthly outgoings.
Overall Progress ("Sampurna Pragati"): A circular or linear progress bar showing % paid off vs total borrowed.
List Tab (Sakriya Rina): Cards for individual shared loans (e.g., "Car Loan", "Home Loan"). Each card displays the Bank name, total amount, remaining balance, and a "Bhugtan Karein" (Make Payment) button.
Primary Action (FAB): '+' icon to "Add Shared Loan" (Sajha Rina Jodein).
Make Payment Modal (Popup mockup): Select loan dropdown, amount input, date, and confirm button.
6. Shared Lists Screen (Soochi)
Dashboard view: Grid or horizontal scroll of list folders (e.g., "Groceries", "IKEA wishlist", "Weekend Plans"). Each folder shows "3/10 completed".
Detail view: A checklist UI. Items can be checked off (which strikes them through and moves them to the bottom).
Primary Action (FAB): '+' icon to add a new list or list item.
7. Shared Memories Screen (Smriti)
Gallery Grid: A 2-column or 3-column masonry grid of photos.
Photo Cards: Each image has subtle rounded corners. When expanded, shows the uploader's name, a caption, and the date.
Primary Action (FAB): Camera or Image '+' icon to upload a new memory.
8. Profile & Settings (Parichay)
User Header: Large Avatar, Name, Email.
Loyalty Section: "Kutumb Nishtha Vatika" (Loyalty Rewards Dashboard) card showing current loyalty level, total points accrued, and a button to view reward history.
Settings Toggles (List with Switches):
"Enable Expenses Module"
"Enable Loans Module"
"Dark Mode" (Options: System, Light, Dark)
Customization choices: "Theme Color" picker (row of color circles to change the app's primary UI color), "Font Family" dropdown.
Sync & Pair Section: A pairing code or QR code area to pair with a partner/family member.
